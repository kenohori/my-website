Important software

  • azul, a CityGML viewer for macOS (2016-)
  • imbiber, Create nicely formatted HTML from BibTeX files directly from Jekyll (2015-)
  • lcc-tools, tools to construct and manipulate higher-dimensional linear cell complexes (2014-2016)
  • bibhtmler, simple PHP class to convert BiBTeX to HTML (2013-2015)
  • pprepair, (planar partition repair) ensures that a set of polygons form a valid planar partition, made of valid polygons and having no gaps or overlaps (2010-).
  • prepair, (polygon repair) takes a possibly invalid polygon, gives it a consistent interpretation and returns a valid polygon according to the OGC Simple Features and ISO 19107 rules (2010-).
